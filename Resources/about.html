Want to find out how this all works? Or what is planned to be implemented in the future? Then, this is the right place!

Though this site is just in its primary stage and far from completeness, I already want to mention some ideas that are likely to be included in the future:

  • Regarding SkewbSkills:
    • Other practicing modes:
      • Add a One-Look-Trainer (which would be a combination of FLT and L2LT)
      • Recognition Quiz: based on some visible faces, the user has to find out the case; maybe add a countdown timer to it after which the scramble won't be visible anymore
      • Competition Sim: no concrete ideas yet, maybe add atmospheric competition sounds, allow only 4 averages per day, random power of other competitors that determine how into how many rounds you can proceed each day
    • Add a "Previous" button so that you can look up again what case you just had
    • Personalisation: let the use identify weak algs or algs that he / she does not know yet; let user vote how good the recognition / execution was; have a database storing the user's results; let the user add his / her own algs
    • Remove occurences of the same move right after itself (or inverse etc.) which is a bug of the WCA notation function
    • Improve usability of SkewbSkillsWeb: design of buttons, usage of keyboard shortcuts (e.g. spacebar)
  • Add a database containing all algs I use (or others use) with short video sequences showing their execution
  • Add page explaining how SkewbSkills works (web + offline)
